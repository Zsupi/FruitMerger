# FruitMerger

## About

- Homework for game development course at `BUDAPEST UNIVERSITY OF TECHNOLOGY AND ECONOMICS`

## Control

- With `A` and `D` you can move the player horizontally.
- With `Space` you can drop a new fruit.

## Goal
